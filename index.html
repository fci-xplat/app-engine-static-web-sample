FPT Kubernetes Engine

FPT Kubernetes Engine dựa trên nền tảng mã nguồn mở K8S giúp tự động hoá triển khai, nhân rộng và quản lý các ứng dụng container. FPT Kubernetes Engine tích hợp đầy đủ các thành phần: Container Orchestration, Storage, Networking, Security, PaaS cung cấp cho khách hàng môi trường tốt nhất để phát triển và triển khai ứng dụng trên Cloud.

FPT cung cấp Kubernetes Cluster chỉ với vài click. Giúp đơn giản hóa quá trình cài đặt và quản trị hệ thống. Dễ dàng chuyển đổi ứng dụng lên FPT Cloud hoặc giữa các Cloud. FKE hỗ trợ autoscale giúp cluster có thể tự động scale thêm worker nodes. Tăng tài nguyên cho cluster một cách tự động, giúp ứng dụng đáp ứng lượng tải lớn tăng đột ngột.
